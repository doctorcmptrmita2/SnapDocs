# Welcome to RepoDocs

RepoDocs transforms your GitHub repository's markdown files into beautiful, fast documentation sites instantly.

## Why RepoDocs?

- **30 Second Setup** - Connect GitHub, pick a repo, done
- **Edge-Fast** - 10ms response times globally via Redis cache
- **Git-Native** - Your repo is the source of truth
- **Zero Config** - No YAML files, no build steps

## Quick Start

```bash
# 1. Login with GitHub
# 2. Select your repository
# 3. Your docs are live!
```

## Features

| Feature | Description |
|---------|-------------|
| Auto-Sync | Push to GitHub, docs update automatically |
| Versioning | Branch and tag-based version selector |
| Search | Full-text search across all docs |
| Syntax Highlighting | 150+ languages via Shiki |
